# Z80(Z8S180) CPU board

This board is designed for running CP/M.

|||
|:--|:-|
|CPU|Z8S180|
|Memory Size|128KiB|
|CPU Clock|Max. 16MHz|
|Interface|USB|

The USB port is used as a interface to connect with the subsystem that controles a console and disk drives.
